# Factors
## Write a program to print all the factors of a number other than 1 and the number itself.
### Input Format :
A single integer, n
### Output Format :
All the factors of n excluding 1 and the number itself
### Constraints :
```sh
0 <= n <= 10,000
```
### Sample Input 1 :
```sh
8
```
### Sample Output 1 :
```sh
2 4
```

```sh
11
```
